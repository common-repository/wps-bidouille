== Description ==

= Français =

WPS Bidouille fournit des informations sur votre WordPress et contient des outils d’optimisation.

Vous pourrez ainsi tout connaitre de votre serveur, optimiser certains réglages et télécharger un rapport TXT de ces valeurs.

* Masquer les erreurs de connexion
* Retirer les caractères spéciaux des medias téléversées
* Retirer la version de WordPress
* Supprimer le manifeste de Windows Live Writer
* Désactiver les emoji WordPress
* Restreindre le nombre de révisions
* Retirer H1 dans Tiny MCE
* Désactiver l'API REST
* Ajouter le format d'image “Medium Large”
* Désactiver la page auteur et le lien auteur
* Désactiver les utilisateurs dans le sitemap WordPress par défaut
* Supprimer le flux RSS
* Supprimer le flux RSS des commentaires
* Charger la dernière version de jQuery
* Améliorer la vitesse de WooCommerce
* Améliorer la vitesse de Contact Form 7
* Désactiver les endpoints utilisateur de l'API REST

Ainsi que des outils complémentaires :

* Déconnecter toutes les sessions
* Supprimer les données temporaires expirées
* Nettoyer les commentaires indésirables
* Vider les commentaires de la corbeille

Vous pourrez également réparer et optimiser votre base de données directement depuis WPS Bidouille.

Enfin, WPS Bidouille vous propose un centre de notifications amélioré permettant en un clin d’oeil de voir ce qu’il convient de mettre à jour, ce qu’il faut supprimer, corriger ou ajuster.

WPS Bidouille provides information about your WordPress and contains optimization tools.

You will be able to know everything about your server, optimize certain settings and download a TXT report of these values.

* Hide connection errors
* Remove special characters from uploaded media
* Remove the version of WordPress
* Remove Windows Live Writer Manifest
* Disable WordPress emoji
* Restrict revision number
* Remove H1 in Tiny MCE
* Disable the REST API
* Add the image format "Medium Large"
* Disable author page and author link
* Disable users in default WordPress sitemap
* Delete the RSS feed
* Remove RSS feed from comments
* Load the latest version of jQuery
* Improve the speed of WooCommerce
* Improve the speed of Contact Form 7
* Disable REST API user endpoints

As well as complementary tools:

* Disconnect all sessions
* Delete expired temporary data
* Clean up unwanted comments
* Empty comments from trash

You can also repair and optimize your database directly from WPS Bidouille.

Finally, WPS Bidouille offers you an enhanced notification center that allows you to see what needs to be updated, what to delete, correct or adjust.

To learn more read the following article: <a href="https://wpformation.com/wps-bidouille/" target="_blank">https://wpformation.com/wps-bidouille</a>

- <a href="https://wordpress.org/plugins/wps-cleaner/" target="_blank">WPS Cleaner</a> to clean your WordPress site.

== Installation ==

= Français =
1. Aller dans Extensions › Ajouter.
2. Rechercher *WPS Bidouille*.
3. Recherchez ce plugin, téléchargez-le et activez-le.

= English =
1. Go to Plugins › Add New.
2. Search for *WPS Bidouille*.
3. Look for this plugin, download and activate it.
